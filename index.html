About

I recording some hikes starting in 2018, and then started recording almost all hikes in summer 2020. This page contains data about all of my recorded hikes, including statistics and pictures. I made these web pages with a package I developed, trailTracker. The maps are made using Leaflet, and also rely on modified versions of Leaflet GPX and EXIF to GeoJSON. On each hike's page, the data are processed in Python primarily using gpxpy and are plotted using Plotly.
